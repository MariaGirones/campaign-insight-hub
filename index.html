Campaign Hub Dedupe Duplicate Record Finder

How it works

  1. Drop a PDF or CSV file below — PDF is preferred for best results.
  2. Your file must be a list where the first row contains the column names (e.g. Name, Email, Instagram).
  3. Select the columns you want to scan, then hit Find Duplicates.

Note Row 1 is always your header row. Data rows are numbered from 2 onwards — that's how row numbers appear in the results.
